## Housing Data Cleaning Pipeline

A data cleaning project that merges and preprocesses four regional housing datasets into a single, analysis-ready CSV file using Python and pandas.

---

## Project Overview

Raw housing data sourced from [Redfin](https://www.redfin.com) is often inconsistent, incomplete, and spread across multiple files. This notebook demonstrates a full end-to-end cleaning pipeline, including:

- Merging multiple CSV files into one combined dataset
- Dropping irrelevant or redundant columns
- Handling missing values using appropriate strategies per column type
- Computing derived fields from existing data
- Exporting a clean, ready-to-use CSV
